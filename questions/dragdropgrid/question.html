The question below was designed to showcase the different features of pl-grid. This element creates a 2-dimensional grid of blocks that can be filled by students via drag and drop. The element is auto-graded based on the number of blocks that match the sample solution (excluding pre-filled blocks). It was primarily designed for questions related to trace tables (see the other example questions), but can also be used more broadly when an answer in the form of a 2D grid is required.

Element Demo

Below is an example of a generic question where objects have to be sorted into a specific row and column of the grid. The attribute color-column is set to true, so blocks that are placed in a column are automatically colored to match the pre-filled blocks. Pre-filled blocks cannot be edited, allowing the placement of guideline blocks that are not part of the student solution. The width and height of blocks can be customized, which is not used in this example, but in the other example questions.
